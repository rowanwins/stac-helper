ItemCollection Class

Works with a STAC Item Collection , which is returned from searches or when paginating through items of a STAC Collection.

Constructor Arguments

NameTypeDefaultDescription
jsonjsonThe raw json of the stac item
parent(Collection|Search|null)nullThe parent class that created this, either a Collection or a Search.
prevItemCollection(ItemCollection|null)nullThe previous Item Collection from the same parent
nextItemCollection(ItemCollection|null)nullThe next Item Collection from the same parent
linkThatCreatedThis(json|null)nullThe link object that was used to retrieve the json for this item collection.

Properties

NameReturnsDescription
rawJson
jsonThe json of the stac entity
parent
(StacEntity|Search|null)The parent stac entity class
pageItems
Array <Item>An array of Item objects created from the STAC Items
linkThatCreatedThis
(JSON|null)The JSON Link Object that was used to retrieve the data for this ItemCollection. This will often be the next or prev link from another page.
stacType
stringGet's a string of the stac type as 'ItemCollection'
prevLink
JSONReturns the "prev" or "previous" rel StacLink if it exists, or null
hasPrevLink
booleanReturns a boolean indicating if the ItemCollection has a previous page link
nextLink
JSONReturns the "next" rel StacLink if it exists, or null
hasNextLink
booleanReturns a boolean indicating if the ItemCollection has a next page link
